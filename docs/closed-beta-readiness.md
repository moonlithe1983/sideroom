# SideRoom Closed Beta Readiness

Date: 2026-04-13
Status: pending

## Must Be True Before Inviting Beta Testers

- Live Supabase preview or staging backend is configured and validated.
- Email and Google auth both work on real Android devices.
- Seed content prevents first-run emptiness.
- Accessibility checklist has been run on real devices.
- Real-device smoke pass has been completed on the current preview build.
- Moderation drills have been completed with moderator and restricted accounts.
- Policy, support, privacy, and account-deletion request pages are live.
- No known P0 or P1 bugs remain.

## Beta Ops Checklist

- Tester group is chosen.
- Support contact is monitored.
- Incident-response owner is assigned.
- Rollback plan exists for a bad preview build.
- Feedback capture path is ready.

## Exit Rule

Change `Status: complete` only when the list above is true and the team is ready to let strangers judge the app.
